# Sky7 Corporate Ecosystem

A unified digital connected ecosystem platform linking manufacturers, distributors, dealers, retailers, business partners, and entrepreneurs.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the application locally in development mode:
   ```bash
   npm run dev
   ```

## Production Build

To build the static assets for deployment:
```bash
npm run build
```
The optimized bundle will be generated inside the `dist/` directory, ready to be hosted on Vercel, Netlify, or any static hosting provider.
